# Elarning

## Prerequirement
- docker
- docker-compose

## Moodle
- Run command
```
# docker-compose -f moodle/4.2.yml up
```
- User and Site configuration
  - http://localhost:8080/
  - MOODLE_USERNAME: Moodle application username. Default: user
  - MOODLE_PASSWORD: Moodle application password. Default: bitnami
